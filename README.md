# 🚀 R Vikranth — Portfolio

A sleek, dark-themed personal portfolio built with **HTML, CSS & JavaScript** — featuring smooth scroll animations, a typing hero effect, and a fully responsive layout.

🔗 **Live:** [rapakavikranth.vercel.app](https://rapakavikranth.vercel.app)

---

## ✨ Features

- **Animated Hero** — gradient text with typing role rotation
- **Interactive Timeline** — education history with pulsing markers
- **Project Showcase** — hover overlays linking to GitHub repos
- **Contact Form** — floating-label inputs with focus glow
- **Cursor Glow** — subtle mouse-follow light effect (desktop)
- **Fully Responsive** — optimized for mobile, tablet & desktop

## 🛠️ Tech Stack

| Layer     | Tools                                           |
| --------- | ----------------------------------------------- |
| Structure | HTML5, Semantic Elements                        |
| Styling   | Vanilla CSS, CSS Variables, Keyframe Animations |
| Logic     | Vanilla JavaScript (ES6+)                       |
| Hosting   | Vercel                                          |
